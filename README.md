# Team Ragnar

A springboot REST API project for managing an online book store (Backend)

# Features
- User and Admin Registration
- CRUD for Books and Library management
- View bookings

# Concepts Covered:
- Entity Relationships
- DTOs
- Role-based access control (user, admin)
- Integration Testing (Controller layer)
- Spring Security
- CRUD Operations
- REST + Database integration
- Exception Handling
